1. 这个项目是一个基于 Vue3 + TypeScript + FastAPI + Langchain 的简历助手项目，用于分析简历并给出对应的简历更改指导以及给出投递适合岗位的建议。(主要使用模块化设计)
2. 这个项目的主要功能包括：
   - 简历分析：用户上传简历文件pdf文件格式或者docx文件格式，项目会分析简历内容，给出简历更改指导以及合适的岗位投递建议。
   - 岗位建议：用户输入期望岗位名称，项目会根据期望的岗位名称，给出符合期望岗位要求的简历更改建议。
   - AI对话：用户可以在项目中与AI进行对话，询问相关问题，AI会根据用户的问题，给出对应的建议。如果用户已经进行过简历分析和岗位推荐，AI会根据用户的简历分析结果和岗位推荐结果给出建议。
3. 这个项目的技术栈包括：
   - 前端：Vue3 + TypeScript + ElementPlus
   - 后端：FastAPI (Version 0.136.0)+ Langchain (Version 1.2.15) + zai-sdk (Version 0.2.2)
   - 数据库：关系型数据库（MySQL） + 向量数据库（ChromaDB）
   - Python包管理器：UV
4. 代码生成或编辑规则：
   - 所有Python代码必须符合RESTful API设计风格规范。
   - 所有的代码都必须符合项目的模块化设计。
